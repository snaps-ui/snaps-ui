## Quickstart

### Snaps UI is a modern React component library built on top of **Ark UI** and **Panda CSS**.

This guide will walk you through installing and configuring it in your project.

## Installation

### Install Snaps UI and its peer dependencies:

```bash
# npm
npm install @snaps-ui/react @snaps-ui/panda-preset @pandacss/dev

# pnpm
pnpm add @snaps-ui/react @snaps-ui/panda-preset @pandacss/dev

# yarn
yarn add @snaps-ui/react @snaps-ui/panda-preset @pandacss/dev

# bun
bun add @snaps-ui/react @snaps-ui/panda-preset @pandacss/dev
```

## Setting up Panda CSS

### If Panda isn‚Äôt initialized yet, run the following command:

```bash
npx panda init --postcss
```

```ts
// panda.config.ts
import { defineConfig } from '@pandacss/dev'
import { createSnapsPreset } from '@snaps-ui/panda-preset'

export default defineConfig({
  presets: [createSnapsPreset],
  include: ['./src/**/*.{js,jsx,ts,tsx,mdx}'],
  importMap: '@snaps-ui/styled-system',
  outdir: 'styled-system',
  jsxFramework: 'react',
})
```

### Then generate your styles by running:

```bash
npm run panda:codegen
```

### If you don‚Äôt already have a script for this, add one to your package.json:

```json
{
  "scripts": {
    "prepare": "panda codegen"
  }
}
```

### This will automatically regenerate styles every time you install dependencies.

### Using Components

### Once setup is complete, you can start using Snaps UI components immediately:

```tsx
import { Button } from '@snaps-ui/react/button'

export default function Example() {
  return <Button>Click me</Button>
}
```

## Don't forget to add this to your style

```css
@layer reset, base, tokens, recipes, utilities;
```

### Snaps UI components are theme-aware, lightweight, and designed for accessibility out of the box.

### Customizing the Theme

You can extend the default Snaps UI theme directly in your Panda configuration:

```ts
// panda.config.ts
import { defineConfig } from '@pandacss/dev'
import { createSnapsPreset } from '@snaps-ui/panda-preset'
import neutral from '@snaps-ui/panda-preset/colors/neutral'

export default defineConfig({
  presets: [
    '@snaps-ui/panda-preset',
    createSnapsPreset({
      accentColor: neutral,
      grayColor: neutral,
      radius: 'sm',
    }),
  ],
  importMap: '@snaps-ui/styled-system',
  preflight: true,
  include: ['./app/**/*.{js,jsx,ts,tsx}', './components/**/*.{js,jsx,ts,tsx}'],
  staticCss: {
    recipes: '*',
  },
})
```
