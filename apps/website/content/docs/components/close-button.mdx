---
title: Close Button
description: A button used to close modals, drawers, alerts, and other dismissible elements.
---

<ExampleTabs name="close-button" scope="close-button" />

## Usage

```tsx
import { CloseButton } from '@snaps-ui/react'

;<CloseButton />
```

It accepts all props available in **Button** component.

## To change the default icon, wrap your custom icon as a child:

```tsx
<CloseButton>
  <YourCustomIcon />
</CloseButton>
```
