```tsx
;<Spinner />
```

## Props

| Prop           | Type                                   | Default          | Description                       |
| -------------- | -------------------------------------- | ---------------- | --------------------------------- |
| `size`         | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"`           | Controls the size of the spinner. |
| `colorPalette` | `string`                               | `"currentColor"` | Sets the color of the spinner.    |
