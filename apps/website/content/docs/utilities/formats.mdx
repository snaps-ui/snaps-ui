## Usage

The **@snaps-ui/react** package provides simple formatters to make displaying data more readable — such as converting file sizes, numeric values, and timestamps.

```tsx
import { FormatBytes, FormatNumber, FormatRelativeTime } from '@snaps-ui/react'
```
